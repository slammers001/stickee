# Stickee v0.0.1 - Initial Release

## Overview
**Stickee** is a modern desktop application for managing and organizing digital sticky notes and tasks. Built with Electron, React, and TypeScript, it provides a delightful note-taking experience with the look and feel of real sticky notes.

## Features
- **Digital Sticky Notes**: Create colorful notes with authentic sticky note aesthetics
- **Task Management**: Organize notes with status tracking (To-Do, Doing, Done, Backlog)
- **Color Coding**: 10 vibrant colors to choose from (yellow, pink, blue, green, purple, orange, teal, lavender, peach, mint)
- **Pinning System**: Pin important notes to keep them at the top of your board
- **View Modes**: Switch between grid and list layouts for different viewing preferences
- **Modern UI**: Clean, professional interface built with shadcn/ui components
- **Cross-Platform**: Desktop application available for Windows

## Technical Details
- **Frontend**: React 18 with TypeScript
- **Desktop**: Electron framework
- **Styling**: Tailwind CSS with custom note color schemes
- **UI Components**: Radix UI + shadcn/ui
- **State Management**: React hooks with TanStack Query
- **Build Tools**: Vite for fast development and building
- **Animations**: Smooth transitions and hover effects for enhanced UX

## What Makes Stickee Special
- Realistic sticky note appearance with random rotation and shadow effects
- Handwriting-style font for authentic note feel
- Responsive design that works on different screen sizes
- Intuitive drag-and-drop interface
- Local storage for offline access
- Clean, minimalist design that focuses on productivity

## Installation
Download the installer from the releases page and follow the installation wizard.

## Usage
1. Launch Stickee from your desktop
2. Click the + button to create your first note
3. Choose a color and status for your note
4. Start organizing your thoughts and tasks!
