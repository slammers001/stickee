═══ STICKEE — New Feature Roadmap ═══

Stickee is a React + Tauri + Supabase sticky-note app. A "feature" is a major
named capability on par with StickeeList (checklists) or Speak-to-Text (voice).
Already built: StickeeList, Speak-to-Text, StickeeAuth, StickeeFonts,
StickeeTheme, StickeeArchive, StickeeExport, StickeeReactions, StickeeSearch,
StickeeQuick, StickeeDrag, StickeeIssue.

═══════════════════════════════════════
 TOP 10 MOST VALUABLE NEW FEATURES
═══════════════════════════════════════

1.  StickeeAI — 4-5 weeks
    AI assistant integrated into every note. Auto-summarize long notes, suggest
    tags, rewrite/improve text, expand bullet points into paragraphs, and answer
    questions about your notes. Uses OpenAI API or local LLM with streaming
    responses shown inline.

2.  StickeeBoard — 3-4 weeks
    Kanban board view that organizes notes into columns by status
    (To-Do / Doing / Done / Backlog). Drag notes between columns to change
    status. Built on existing drag-and-drop infrastructure. Toggle between
    grid view and board view.

3.  StickeeShare — 3-4 weeks
    Share notes via public link (read-only) or invite specific users by email
    (collaborative edit). Shared notes show live cursors and real-time changes
    via Supabase Realtime. Permission levels: view, comment, edit.

4.  StickeeMobile — 8-10 weeks
    Native iOS and Android app built with React Native / Expo. Share the
    Supabase backend, type definitions, and business logic. Touch-optimized
    sticky-note interactions, swipe-to-archive, haptic feedback.

5.  StickeeRemind — 2-3 weeks
    Set date/time reminders on any note. Desktop notifications via Tauri
    notification plugin and Web Notification API. Optional email/SMS reminders
    via Supabase Edge Functions + Twilio/Resend. Snooze and recurring options.

6.  StickeeConnect — 5-6 weeks
    Third-party integrations: Slack (post notes to channels, create notes from
    messages), Discord (webhook), Todoist/TickTick (two-way sync), Notion
    (export), and Google Calendar (create events from notes). OAuth flows
    stored per-user.

7.  StickeeCanvas — 5-6 weeks
    Infinite canvas/whiteboard mode where notes are freely positioned,
    connected with arrows/lines, and zoomable/pannable. Great for mind
    mapping, brainstorms, and visual planning. Uses a canvas library like
    tldraw or Fabric.js.

8.  StickeeClip — 3-4 weeks
    Browser extension (Chrome + Firefox) that clips selected text, full
    articles, screenshots, or entire pages into new Stickee notes. Right-click
    context menu + toolbar icon. Works with Tauri desktop app via deep links
    or direct Supabase API calls.

9.  StickeePages — 3-4 weeks
    Wiki-like linked notes. Create internal links between notes using [[wiki]]
    syntax, with backlinks shown in a sidebar. Browsable graph view of
    connected notes. Builds on existing markdown/react-markdown infrastructure.

10. StickeeOffline — 4-5 weeks
    Full offline mode: notes cached in IndexedDB via Dexie.js, writes queued
    and synced when online, conflict resolution with last-write-wins or
    interactive merge. Service worker for the web build. Makes Stickee usable
    on planes, trains, and spotty connections.

═══════════════════════════════════════
 NEXT 5 WORTHY FEATURES (11–15)
═══════════════════════════════════════

11. StickeeLock — 3-4 weeks
    Proper end-to-end encryption using Web Crypto API (AES-256-GCM) with a
    user-set passphrase. A separate "vault" for sensitive notes. Biometric
    unlock on mobile. Current base64 obfuscation replaced with real crypto.

12. StickeeWidget — 2-3 weeks
    Desktop widgets for Windows (Widgets Board), macOS (Notification Center
    widgets), and Android/iOS home screen widgets. Show pinned notes,
    reminders due today, and quick-add button. Uses Tauri plugins and native
    widget APIs.

13. StickeeTemplates — 2-3 weeks
    Note templates for common use cases: meeting notes, project plan,
    daily journal, weekly review, bullet journal, gratitude log, habit
    tracker. Users can create and save their own templates too.

14. StickeeCal — 2-3 weeks
    Calendar view that surfaces notes by creation date, due date (from
    StickeeRemind), or a custom note date. Drag notes onto dates. Month/week/day
    views. Integrated with Google Calendar export.

15. StickeeFlow — 3-4 weeks
    Automation engine: "If I add a note with status Done, move it to Archive
    after 7 days" or "If a note is tagged 'shopping', add it to my Todoist."
    Visual trigger-action builder. Runs on Supabase Edge Functions + cron.

═══════════════════════════════════════
 NOTES
═══════════════════════════════════════
- Estimates assume one senior full-stack developer.
- StickeeAI and StickeeOffline have the highest user impact.
- StickeeBoard is the quickest high-value win (builds on existing code).
- StickeeMobile is the longest but unlocks the biggest new audience.
- StickeeConnect and StickeeFlow share integration infrastructure.
