# Stickee v0.0.3 Release Notes 🎉

## 5 New Features ✨

### 🔍 Search Functionality
- Added search bar with real-time filtering
- Search works across note content, titles, status, and color
- Search results update instantly as you type

### 📝 Note Titles
- Optional title field for each note
- Titles display in bold above note content
- Limited to 12 characters with truncation
- Works in both grid and list views

### ⌨️ Keyboard Shortcuts
- Press 'n' to create new note
- Press 'p' to open quick note popup
- Shortcuts work when not typing in input fields

### 🪟 Quick Note Popup
- Separate window popup for quick note creation
- Press 'p' to open minimal sticky note window
- Auto-saves when closed or with Esc/Ctrl+Enter
- Uses different styling as main app to show it's a popup
- Basic popup styling

### 🔐 Note Encryption
- Note content and titles better encrypted
- Automatic encryption/decryption for seamless user experience


### 🚀 Improvements
- New notes now appear first (newest first ordering)
- Enhanced privacy and security
- Better user experience with keyboard shortcuts
- Consistent styling across all components

## ⚙️ Technical Details
- Updated service layer for encryption
- Enhanced keyboard event handling
- Improved window management for popup notes
- Better error handling and user feedback
