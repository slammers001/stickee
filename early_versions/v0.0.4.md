# Stickee v0.0.4 Release Notes 🎉

## 🆕 Major Features

### 🎨 74+ Fonts Added
- **4 Title Fonts** for note headings
- **28 Basic Fonts** for clean, readable text
- **39 Handwriting Fonts** for personal style
- **3 Standard System Fonts**

### ✨ Font Management
- **Favorite Fonts** - Bookmark up to 10 favorites
- **Font Previews** - See how each font looks before applying
- **Cloud Sync** - Font preferences saved to your account
- **Responsive Loading** - Fonts load efficiently

### 🎯 Title Fonts (New!)
- **Arbutus** - Bold, decorative serif
- **Agbalumo** - Playful, rounded display
- **Walter Turncoat** - Casual handwriting
- **Yatra One** - Elegant, flowing script

### 🔤 Basic Fonts (28 New)
- **Sans-serif**: ABeeZee, Aclonica, Advent Pro, Tenali Ramakrishna, Truculenta, Unbounded
- **Monospace**: Anonymous Pro, Ubuntu Sans Mono, Nova Mono, Syne Mono, VT323, Xanh Mono, Cutive Mono
- **Serif**: Arbutus Slab, Nixie One, Noticia Text, Arvo, Oi, Oldenburg
- **Display**: Orelega One, Nova Oval, Atma, Butcherman, Cherry Bomb One

## 🛠️ Technical Improvements
- New Font Settings Service with Supabase integration
- Optimized font loading for better performance
- Responsive font previews in settings
- Memory-efficient font management

## 🔄 How to Use
1. Open Settings (gear icon)
2. Go to "Fonts" tab
3. Choose from Basic or Handwriting fonts
4. Use the "Titles" tab to set heading fonts
5. Click the ❤️ icon to save favorites
